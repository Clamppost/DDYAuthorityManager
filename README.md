# DDYAuthManager

* 各种权限申请(有的不支持二次申请，比如联网权限，在国行版iOS10+一个bundleID只会询问一次)
* 各种权限判断（只有有权限才能下一步操作，例如拍照）

联网权限不弹窗或者wifi下弹窗拒绝蜂窝网络问题 见https://github.com/Zuikyo/ZIKCellularAuthorization
